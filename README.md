# Hippocampus – Content Management Platform

A full-stack web application that allows users to **save, organize, and manage content from multiple platforms in one centralized dashboard.**

---

## Backend Repository

Backend Code:
👉 **[Hippocampus Backend](https://github.com/JalajKumarPrajapati/Hippocampus_Backend)**

---

## Features

* Save content from **Twitter/X, YouTube, LinkedIn, and Google Maps**
* **Automatic content detection** from URLs with native embedding
* **Mobile-responsive UI** with smooth navigation
* **Platform-based filtering** to quickly view specific content
* **Built-in sharing options**
* **Secure JWT authentication** with protected routes
